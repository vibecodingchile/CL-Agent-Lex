## 🚀 Overview
**CL-Agent-Lex** is a technical extension of the [Harvard LIL Agent Protocols](https://harvard-lil.github.io/agent-protocols/). 

While the original protocol focuses on technical interoperability, **CL-Agent-Lex** introduces a **Jurisdictional Layer** for Civil Law systems. Developed by **Matias Rojas Faundez** at **Vibe Coding Chile**, this project transforms AI interactions into legally binding acts under Chilean Law (Ley 19.799 & Ley 21.096).

## 🛠 Key Features
- **Jurisdictional Handshake:** Injects Chilean RUT and FEA (Advanced Electronic Signature) into JSON-RPC calls.
- **On-Chain Attribution:** Solidity smart contracts (`AgentNotary.sol`) that act as digital notaries for agent spending limits.
- **Compliance-as-Code:** Automated validation against Chilean AI Policy 2024-2031.
